# 中医植物大战僵尸（MVP）

运行方法：

1. 打开 index.html 在浏览器中（直接双击或用本地服务器）。
2. 或者把仓库部署到 GitHub Pages（Settings -> Pages -> 选择 main分支 / 根目录）。

玩法：
- 点击上方的植物卡牌选择植物（人参：射手，艾草：阻挡，枸杞：产气）。
- 在画布上点击或触控放置植物，放置会消耗“气”。
- 点击“开始”开始敌人入侵。
- 敌人穿过左侧则游戏失败。

说明：这是一个非常精简的原型版本（MVP），代码结构便于后续扩展：
- 增加关卡、平衡、音效、美术资源
- 改善AI、碰撞检测和性能
